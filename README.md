## ⚡ Cài đặt 1 lệnh

SSH vào Raspberry Pi và chạy:

```bash
sudo bash -c "$(curl -fsSL https://raw.githubusercontent.com/ngocquy020196/RaspberryPi-Proxy-Box/main/setup.sh)" || { git clone https://github.com/ngocquy020196/RaspberryPi-Proxy-Box.git /opt/dcom-proxy && cd /opt/dcom-proxy && sudo bash setup.sh; }
```

Hoặc chạy từng bước:

```bash
git clone https://github.com/ngocquy020196/RaspberryPi-Proxy-Box.git /opt/dcom-proxy
cd /opt/dcom-proxy
sudo bash setup.sh
```

✅ Xong! Script tự động cài tất cả, tạo mật khẩu, khởi động services, và hiện link dashboard.

## 📋 Tính năng

| Tính năng | Mô tả |
|-----------|--------|
| 🌐 **Multi-Proxy** | Mỗi USB Dcom = 1 proxy riêng (HTTP + SOCKS5) |
| 🔄 **Xoay IP** | Đổi IP 4G bằng 1 click (HiLink API / AT Commands) |
| 🖥️ **Dashboard** | Giao diện web dark theme, quản lý thiết bị trực quan |
| 🔒 **Bảo mật** | Đăng nhập bằng secret key, proxy auth per-device |
| 🌍 **Remote Access** | Cloudflare Tunnel — truy cập từ bất kỳ đâu |
| 🔌 **Plug & Play** | Cắm Dcom vào → tự nhận, tự chuyển mode |
| ⚙️ **Auto-start** | Systemd services — tự chạy khi khởi động Pi |

## 🖥️ Phần cứng

- **Raspberry Pi** (3B+/4/5) — chạy Raspberry Pi OS / Debian
- **USB Hub** có nguồn phụ
- **USB Dcom 4G**: Vodafone K5160 / Huawei E3372 (HiLink hoặc Stick mode)

## 📡 Cách sử dụng

### 1. Truy cập Dashboard

Sau khi cài đặt, script sẽ hiện:
```
📡 Dashboard: http://<PI_IP>:8080
🔑 Mật khẩu:  <auto-generated>
```

### 2. Kết nối Proxy

```bash
# HTTP Proxy
curl -x http://proxyuser:proxypass@<PI_IP>:10000 https://api.ipify.org

# SOCKS5 Proxy
curl --socks5 proxyuser:proxypass@<PI_IP>:11000 https://api.ipify.org
```

### 3. Xoay IP

- Bấm nút 🔄 trên Dashboard cho từng Dcom
- Hoặc gọi API: `POST /api/rotate/<interface>`

## ⚙️ Cấu hình

File `.env` (tự tạo khi chạy setup):

| Variable | Mô tả | Mặc định |
|----------|--------|----------|
| `SECRET_KEY` | Mật khẩu dashboard | Auto-generated |
| `PORT` | Port web dashboard | `8080` |
| `PROXY_START_PORT` | Port bắt đầu proxy | `10000` |
| `IP_ROTATE_METHOD` | Phương thức xoay IP | `hilink` |
| `DEFAULT_PROXY_USER` | Username proxy | `proxyuser` |
| `DEFAULT_PROXY_PASS` | Password proxy | `proxypass` |

## 🛠️ Quản lý

```bash
# Xem status
sudo systemctl status dcom-proxy

# Xem logs
sudo journalctl -u dcom-proxy -f

# Restart
sudo systemctl restart dcom-proxy
sudo systemctl restart 3proxy

# Đổi mật khẩu
nano /opt/dcom-proxy/.env    # Sửa SECRET_KEY
sudo systemctl restart dcom-proxy
```
