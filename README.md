## ⚡ One-Command Install

SSH into your Raspberry Pi and run:

```bash
sudo bash -c "$(curl -fsSL https://raw.githubusercontent.com/ngocquy020196/RaspberryPi-Proxy-Box/main/setup.sh)"
```

Or install step by step:

```bash
git clone https://github.com/ngocquy020196/RaspberryPi-Proxy-Box.git /opt/dcom-proxy
cd /opt/dcom-proxy
sudo bash setup.sh
```

✅ Done! The script automatically:
1. Installs Node.js, 3proxy, cloudflared, usb-modeswitch
2. Generates a random password & starts all services
3. **Asks if you want to configure Cloudflare Tunnel** (remote access)
4. Displays dashboard URL + login credentials

| 🌐 **Multi-Proxy** | Each USB Dcom = 1 dedicated proxy (HTTP + SOCKS5) |

## 📡 Usage

### 1. Access Dashboard

After installation, the script will display:
```
📡 Dashboard (LAN):  http://<PI_IP>:8080
🌍 Dashboard (WAN):  https://proxy.yourdomain.com
🔑 Password:         <auto-generated>
```

### 2. Connect via Proxy

```bash
# HTTP Proxy
curl -x http://proxyuser:proxypass@<PI_IP>:10000 https://api.ipify.org

# SOCKS5 Proxy
curl --socks5 proxyuser:proxypass@<PI_IP>:11000 https://api.ipify.org
```

### 3. Rotate IP

- Click the 🔄 button on Dashboard for each Dcom
- Or call API: `POST /api/rotate/<interface>`

## 🌍 Remote Access

### Cloudflare Tunnel (Built into setup.sh)

The installer will **ask if you want to configure Cloudflare Tunnel** at the final step.

If you choose **Yes**, you'll just need to:
1. Login to Cloudflare (opens a browser link)
2. Enter your subdomain (e.g., `proxy.yourdomain.com`)

The script will automatically create the tunnel, route DNS, generate `config.yml`, and install as a service.

**If you skipped during installation**, run manually:

```bash
cloudflared tunnel login
cloudflared tunnel create dcom-proxy
cloudflared tunnel route dns dcom-proxy proxy.yourdomain.com

# Create config
cat > ~/.cloudflared/config.yml << EOF
tunnel: <TUNNEL_ID>
credentials-file: /root/.cloudflared/<TUNNEL_ID>.json
ingress:
  - hostname: proxy.yourdomain.com
    service: http://localhost:8080
  - service: http_status:404
EOF

# Run as service
sudo cloudflared service install
sudo systemctl enable cloudflared
sudo systemctl start cloudflared
```

### Tailscale (Alternative)

```bash
curl -fsSL https://tailscale.com/install.sh | sh
sudo tailscale up
# Use Tailscale IP as proxy host
```

## ⚙️ Configuration

The `.env` file is auto-generated during setup:

| Variable | Description | Default |
|----------|-------------|---------|
| `SECRET_KEY` | Dashboard login password | Auto-generated |
| `PORT` | Web dashboard port | `8080` |
| `PROXY_START_PORT` | Starting port for proxies | `10000` |
| `IP_ROTATE_METHOD` | IP rotation method | `hilink` |
| `DEFAULT_PROXY_USER` | Default proxy username | `proxyuser` |
| `DEFAULT_PROXY_PASS` | Default proxy password | `proxypass` |
